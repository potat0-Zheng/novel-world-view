地形类型体系 — 初版 + 扩展预留

分三级：基础型（必须存在）→ 派生型（增强表现力）→ 自定义型（未来扩展）。所有类型可在编辑器中通过单元格圈选设置

🌊 基础型 — MVP 必做（4 种）

🌊
海洋
深海 / 近海自动区分
🌿
平原
默认大陆地形
⛰
山脉
高海拔，不可通行
🌲
森林
密林区域

🟡 派生型 — 增强表现力（6 种，可按需实现）

🏖
沙滩/浅滩
海洋↔平原过渡带
自动生成，无需手动
🏜
沙漠
干旱区域
🏔
雪原/冻土
极寒地带
🪵
丘陵
平原↔山脉过渡带
适合建筑/城镇
🌋
火山
特殊地貌
🌫
沼泽
湿地/毒雾区

🔮 玄幻扩展型 — 为仙侠/奇幻小说预留（未来迭代）

✨
灵脉/福地
💀
禁地/绝地
🏚
废墟/遗迹
🌌
虚空/异空间
＋ 用户自定义类型（自定义名称 + 颜色 + 图标）
过渡融合示意：相邻地形自然衔接
海洋 → 浅滩 → 平原
平原 → 丘陵 → 山脉
森林 → 平原

扩展性设计：参数化地形定义

// 每个地形类型是一个配置对象，新增只需加一行
const terrainTypes = {
  ocean: { name: '海洋', color: '#1e4080', elevation: -1, walkable: false },
  plain: { name: '平原', color: '#4a7c3f', elevation: 0, walkable: true },
  mountain: { name: '山脉', color: '#4a5a30', elevation: 2, walkable: false },
  forest: { name: '森林', color: '#2d6020', elevation: 0, walkable: true },
  // 后续新增：
  // desert: { name: '沙漠', color: '#c4a43a', elevation: 0, walkable: true },
  // tundra: { name: '冻土', color: '#c8d8e8', elevation: 0, walkable: true },
  // ... 用户自定义类型
}
// 过渡规则：相邻格 elevation 差值 → 自动决定渐变样式
