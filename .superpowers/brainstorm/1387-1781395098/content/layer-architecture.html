依赖规则

L1 → 独立存在
第一图层覆盖所有网格单元。每个格必须是「海洋」或「大陆」。编辑时先完成 L1 才能进入 L2。
L2 → 依赖 L1 大陆
自然地形只能放置在 L1 为「大陆」的格子上。一个格可以叠加多种 L2 类型（如丘陵+森林）。可选——可以没有 L2。
L3 → 依赖 L1 或 L2
建筑必须建于 L1「大陆」或 L2 自然地形之上。不可建在「海洋」。可选——可以没有 L3。

待确认

一个 L2 自然类型是否互斥还是可叠加？比如：「丘陵 + 森林」是否允许并存于同一块大陆格上？（如果是，渲染需要处理层级叠加顺序）
