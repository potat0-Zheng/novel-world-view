模型来源策略

📥 现有模型导入
• Sketchfab / Poly Pizza 等免费 CC0 模型
• 支持 GLTF/GLB 格式导入
• 用户自行下载后拖入模型库
• 系统提供预设的 10-15 个基础模型
🤖 AI 生成模型
• 接入 Meshy AI / Luma AI 等 3D 生成 API
• 用户输入文字描述 → 生成低多边形模型
• 风格统一为 low-poly / 体素风格（降低生成难度）
• 非 MVP 必须，标记为后续迭代

模型风格建议

🔷
Low-Poly 风格
视觉清爽，模型小而快
AI生成友好，个人开发首选
🎨
体素风格
类似 Minecraft，与网格天然契合
但视觉表现力有限
🖌
手绘/风格化
视觉效果最佳
但模型制作成本最高
